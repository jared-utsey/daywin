Google Calendar setup
You need a free Google API key to let DayWin push tasks to your calendar. This takes about 5 minutes and is completely free.
1
Go to console.cloud.google.com and sign in with your Google account.
2
Create a new project (name it anything, like DayWin).
3
Go to APIs & Services → Library, search for Google Calendar API, and enable it.
4
Go to APIs & Services → Credentials, click Create Credentials → API Key. Copy the key it gives you.
5
Also go to OAuth consent screen, create a credential of type OAuth 2.0 Client ID (Web application), add http://localhost and your GitHub Pages URL as authorized origins.
6
Paste your Client ID below and click Connect.
